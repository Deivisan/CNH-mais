# CNH+ Initialize Project

> Inicializa o projeto CNH+ com estrutura completa

## Usage
/initialize-project

## Description
Inicializa o projeto CNH+ criando:
- Estrutura de diretórios completa
- Landing page com links para 3 visualizações
- Páginas Admin, Candidato, Instrutor com dados emulados
- Setup inicial do projeto

## Steps
1. Verificar estrutura existente
2. Criar diretórios necessários
3. Gerar landing page com navegação
4. Gerar páginas de visualização
5. Validar links e funcionamento

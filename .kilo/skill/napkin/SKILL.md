# Napkin - CNH+ Persistent Memory

## About
This is the CNH+ project napkin - persistent memory for mistakes, corrections, and patterns.

## Project Context
- **Project**: CNH+ - Marketplace de Aulas Práticas de Direção
- **Stack**: HTML/CSS/JS (landing pages), Kotlin/Android (app), Bun (backend)
- **Colors**: #1E3A5F (primary), #2E5A8F (primary-light), #4A90D9 (secondary), #87CEEB (accent)
- **3 Views**: Admin, Candidato, Instrutor

## Data
All pages use emulated/mock data for prototyping.

## Rules
- Use Bun as runtime (NOT Node.js)
- Atomic commits per task
- Parallel execution when possible
- Always verify code before moving on

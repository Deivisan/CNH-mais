# Painel do Instrutor - Visão do Instrutor

## Visão Geral

Área privada do instrutor dentro do aplicativo. Diferente do perfil público que o candidato vê.

**Situação:** Instrutor já cadastrado, aprovado e ativo.

---

## Aba 1 - Visão Geral (Dashboard)

Primeira tela que o instrutor vê ao abrir o app.

### Elementos
- [ ] Foto atual do perfil
- [ ] Nome
- [ ] Status: Ativo
- [ ] Nota média ⭐
- [ ] Total de aulas concluídas
- [ ] Horas trabalhadas no app

### Financeiro Resumido
- [ ] Saldo disponível hoje
- [ ] Saldo pendente (aulas ainda não confirmadas)
- [ ] Mensagem: "O saldo é atualizado a cada 24 horas."

### Atalhos Rápidos
- [ ] Editar perfil
- [ ] Ver agenda
- [ ] Ver financeiro
- [ ] Falar com suporte

---

## Aba 2 - Meu Perfil (O que o Candidato Vê)

Visualiza o perfil público exatamente como o candidato vê.

### Pode EDITAR
- [ ] Foto de perfil
- [ ] Biografia
- [ ] Estilo de ensino
- [ ] Especialidades
- [ ] Regiões atendidas
- [ ] Disponibilidade
- [ ] Tipo de veículo:
  - [ ] Carro próprio
  - [ ] Carro do candidato
  - [ ] Moto (se aplicável)

### NÃO pode editar
- [ ] Avaliações
- [ ] Nota média
- [ ] Comentários
- [ ] Horas trabalhadas
- [ ] Quantidade de alunos
- [ ] Selo "verificado"

---

## Aba 3 - Mídia (Vídeos do Instrutor)

### Tipo de Conteúdo Permitido
- [ ] Vídeos curtos mostrando aula
- [ ] Vídeos explicando baliza, controle de embreagem
- [ ] Vídeos de evolução de alunos (sem expor rosto/nome sem autorização)

### Limites
- **Máximo:** 3 a 5 vídeos
- **Duração:** 30-60 segundos
- **Política:** Sempre substitutivo (novo apaga um antigo)

### Fotos
- Opcional
- Limite: até 5
- Mesmo princípio: mostrar trabalho, não álbum pessoal

---

## Aba 4 - Resultados / Desempenho

Acompanha o desempenho.

### Métricas
- [ ] Taxa de aulas concluídas
- [ ] Taxa de cancelamento
- [ ] Avaliação média ao longo do tempo
- [ ] Quantidade de alunos aprovados (se inferido pelo sistema)

**Nota:** "Aprovados" é dado calculado, não editável. Se não der para comprovar, pode ficar oculto.

---

## Aba 5 - Financeiro

### Elementos
- [ ] Saldo disponível
- [ ] Saldo pendente
- [ ] Histórico de repasses
- [ ] Histórico de aulas pagas

### Regras de Liberação
Aula só vira saldo disponível quando:
1. Instrutor marca como concluída
2. Candidato dá OK

### Estado "Pendente"
- Até o OK do candidato, valor fica como pendente

### Frequência de Repasse
- [ ] Diário (a cada 24h)
- [ ] Semanal
- [ ] Mensal

### Mensagem Fixa
> "Os repasses seguem a frequência escolhida e são processados automaticamente."

---

## Aba 6 - Dados Bancários

Gerencia como recebe o dinheiro.

### Pode
- [ ] Cadastrar conta bancária ou Pix
- [ ] Alterar dados bancários

### Regras
1. Conta precisa estar no nome do instrutor
2. Toda alteração: entra em análise
3. Prazo de segurança: 24-48h

### Aviso
> "Alterações bancárias passam por validação por segurança."

---

## Aba 7 - Aulas & Disputas

Resolve o problema do "e se o candidato não der OK?"

### Fluxo Normal
1. Aula acontece
2. Instrutor marca como concluída
3. Candidato confirma (OK)
4. Dinheiro é liberado

### Se Candidato NÃO Confirmar
- [ ] Aula entra como "em análise"
- [ ] Abre-se uma disputa

### Modelo de Disputa
1. Candidato explica o problema
2. Instrutor responde
3. Plataforma analisa:
   - Histórico
   - Pontualidade
   - Mensagens

### Decisões Possíveis
- [ ] Liberar pagamento
- [ ] Reembolso parcial
- [ ] Reembolso total

### Regra Clara
> "O candidato é o carro-chefe. Conduta inadequada pode gerar bloqueio ou exclusão do instrutor."

## 1.4 - Recomendação do Sistema

Com base no perfil e nas respostas, o sistema calcula:
- [ ] Quantidade ideal de aulas
- [ ] Tipo de aula ideal:
  - [ ] Com carro do instrutor
  - [ ] Com instrutor + carro com pedal
  - [ ] Com instrutor sem carro (se candidato tiver carro)

### O candidato vê:
> "Recomendamos X aulas para você iniciar."

### Opções do candidato:
- [ ] Aceitar
- [ ] Reduzir (o sistema guarda o original)

---

## 1.5 - Pagamento

O candidato seleciona:
- [ ] Quantidade de aulas
- [ ] Forma de pagamento:
  - [ ] PIX
  - [ ] Cartão de crédito
  - [ ] Cartão de débito

**Importante:** O dinheiro fica retido no app (não vai direto pro instrutor).

---

## 1.6 - Match com Instrutor

Depois de pagar, o candidato vê:
> "Estamos encontrando o melhor instrutor para o seu perfil."

**O candidato não escolhe o instrutor.** O app decide com base no algoritmo de match.

---

## 1.7 - Primeira Aula

- [ ] Sistema agenda a primeira aula
- [ ] Candidato pode trocar de instrutor **somente após a primeira aula**
- [ ] Depois disso o vínculo é fixo até terminar o pacote

---

## 1.8 - Tela Principal (Home do Candidato)

### Elementos Exibidos
- [ ] Próximas aulas
- [ ] Quantas aulas já fez
- [ ] Quantas aulas faltam
- [ ] Quantas aulas recomendadas originalmente
- [ ] Quantas aulas extras pode ganhar indicando amigos
- [ ] Seu código de indicação
- [ ] Suas recompensas:
  - [ ] aulas extras
  - [ ] seguros gratuitos (se houver)
  - [ ] aulas pós-CNH

---

## Regras Importantes

### Vínculo com Instrutor
- Primeira aula: livre para trocar
- Após primeira aula: vínculo fixo até final do pacote
- Exceção: apenas com aval do suporte em casos graves

### Pagamento
- Dinheiro retido no app
- Liberação apenas após confirmação do candidato
- Dispute disponível se houver problema
